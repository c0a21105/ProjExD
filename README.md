# 第1回
## サザエさんクイズ(ex01/quiz.py)
### 遊び方
* コマンドラインでquiz.pyを実行すると、標準出力に問題が表示される。
* 標準入力から答えを入力する。
* 正解なら「正解！」、不正解なら「不正解...」と表示される。
* 正解でも不正解でも１問のみの出題となる。
* 回答後に回答所要時間が表示される。
### プログラム内の解説
* main関数：クイズプログラム全体の流れの担当する。
* shutudai関数：ランダムに選んだ問題の出題と、回答をmain関数に返す。
* kaitou関数：回答と正解をチェックし、結果と回答所要時間を出力する。

## 消えたアルファベットを探すゲーム(ex01/alphabet.py)
### 遊び方
* コマンドラインでalphabet.pyを実行すると、標準出力に問題が表示される。
* 標準入力から答えを入力する。
* 最大5回のコンティニューが可能になっている。
* 対象文字数は10文字、欠損文字数はランダムで1~5文字になっている。
* 最初に欠損文字数がいくつか聞かれる。
* 欠損文字数があっていれば具体的な欠損文字を聞かれる。
* 欠損文字がすべてあっていればメッセージが表示され、クリア所要時間が表示される。
* どこかで間違っていると、コンティニュー回数が減り、やり直しとなる。
* 5回間違うと、メッセージとともに終了となる
### プログラム内の解説
* play関数：初期設定とゲーム全体の流れを担当する。
* alp_gen関数：10文字のランダムなアルファベットを生成する。
* ask関数：対象文字と表示文字を表示する。
* ans関数：それぞれの回答と正解をチェックし、すべて正しければメッセージと所要時間を表示する。
* main関数：スタート時間の記録と、ゲームを続けるかどうかの判定を行う。
